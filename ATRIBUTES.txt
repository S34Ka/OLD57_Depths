activebackground	- кнопка при наведении курсора
activeforeground	- цвет текста виджета в активном состоянии
anchor				- привязка содержимого внутри виджета ("n", "s", "e", "w", "center", "ne", "nw", "se", "sw")
background			- цвет фона виджета
borderwidth			- толщина границы виджета
cursor				- вид курсора при наведении на виджет (например, "hand2", "cross", "pirate")
disabledforeground	- цвет текста, когда виджет отключен (state="disabled")
font				- шрифт текста
foreground			- цвет фона виджета
highlightbackground	- цвет области вокруг виджета, когда он не в фокусе
highlightcolor		- цвет рамки при фокусе
highlightthickness	- толщина рамки фокуса
image				- изображение (должно быть создано через PhotoImage или Pillow)
justify				- выравнивание текста ("left", "right", "center")
padx				- внутренний отступ по горизонтали
pady				- внутренний отступ по вертикали
relief				- стиль границы (может быть: "flat", "raised", "sunken", "groove", "ridge", "solid")
repeatdelay			- задержка (в миллисекундах) перед началом повторения действия (например, для кнопки, которая должна повторять команду при удержании)
repeatinterval		- интервал (в миллисекундах) между повторениями команды
takefocus			- может ли виджет получать фокус (True/False)
text				- текст, отображаемый на виджете
textvariable		- привязка текста к переменной
underline			- индекс символа в тексте, который будет подчёркнут
wraplength			- максимальная ширина текста перед переносом на новую строку (в пикселях)

command				- функция, вызываемая при активации виджета
compound			- расположение изображения относительно текста ("top", "bottom", "left", "right", "center")
default				- делает кнопку "по умолчанию" (активируется при нажатии Enter)
height				- высота виджета
overrelief			- стиль границы при наведении курсора (работает для кнопок)
state				- состояние виджета ("normal", "disabled", "active")
width				- ширина виджета
